Client#defaults.options

A list of options used in Challenges
Properties:
Name Type Description
ChallengeAutoContinue Boolean Automatically continue through challenge events. Default: true
ChallengeGetFullScore Boolean Always get the full score. Default: false
ChallengeAlwaysCorrect Boolean Always get the questions "correct" in challenges. Default: false
ChallengeUseStreakBonus Boolean Use the answer streak bonuses. Default: false
ChallengeWaitForInput Boolean Wait for answers before continuing to the next question. Default: false
ChallengeScore Number A set score to earn each question. (0-1,500) Default: null
